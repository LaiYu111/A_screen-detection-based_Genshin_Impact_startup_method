## 介绍

原本是要检测整个屏幕是白色的， 但是不想搞那么麻烦。这是因为要考虑屏幕分割，多进程等东西，就怎么简单怎么来。检测电脑屏幕的第0个像素也就是左上角的第一个像素为**白色**则运行该程序。后面说不定会搞监测整个屏幕的方法。
